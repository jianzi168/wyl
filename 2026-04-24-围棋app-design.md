## 三、模块详解

### 3.1 对局模块（Game）

#### 3.1.1 真人对局

| 功能 | 说明 |
|------|------|
| 对局模式 | 同一设备双人对面对弈 |
| 规则引擎 | 中国规则（默认）/ 日本规则 / 应氏规则 |
| 悔棋 | 可设置次数限制（0次 / 1次 / 3次 / 不限） |
| 认输 / 提和 | 支持投子认负和同意提和 |
| 终局数子 | 自动计算胜负，显示终局图 |
| 计时系统 | 自由时间 / 读秒制 / 包干制 |
| 棋谱保存 | 自动保存对局记录 |
| 棋谱回放 | 支持回放、暂停、快进 |
| SGF 导出 | 导出标准格式棋谱 |

#### 3.1.2 AI 对局

| 功能 | 说明 |
|------|------|
| 难度等级 | 6档：10级 · 8级 · 5级 · 2级 · 1段 · 5段 |
| 思考时间 | 快（5秒）/ 中（15秒）/ 慢（30秒） |
| AI 辅助显示 | 实时胜率曲线 + 建议着手（可开关） |
| 复盘模式 | 对局结束后逐步图解 AI 思路 |
| 外部 API | 支持接入用户自备 AI 服务（配置 URL + API Key） |

**难度与 KataGo Elo 对照（参考值）：**

| 等级 | Elo 范围 | 适用场景 |
|------|----------|----------|
| 10级 | 500-1000 | 纯初学者 |
| 8级 | 1000-1500 | 入门刚会吃子 |
| 5级 | 1500-2000 | 初级，会基本死活 |
| 2级 | 2000-2500 | 入门向中级过渡 |
| 1段 | 2500-3000 | 业余1段水平 |
| 5段 | 3500+ | 业余高段水平 |

#### 3.1.3 棋谱管理

- 对局列表：按时间倒序显示，显示对手/难度/结果/日期
- 棋谱详情：回放、胜率曲线图、着手分析
- 收藏功能：收藏重要棋谱
- SGF 导出：分享或保存到本地

### 3.2 教学模块（Teach）

#### 3.2.1 知识体系总览

**主线：棋力等级进度（共 6 阶段，62 个知识点）**

| 阶段 | 等级范围 | 知识点数 | 核心内容 |
|------|----------|----------|----------|
| 启蒙 | 10级 | 8 | 棋盘构造、棋子识别、气的概念、禁入点、基本吃子法（征吃）、劫的基础、胜负判定、棋子连接 |
| 入门 | 8级-6级 | 12 | 枷吃、倒扑、接不归、双吃、征子原理、劫的深入、真眼与假眼、基本死活形（直三、曲三）、围棋礼仪 |
| 初级 | 5级-3级 | 12 | 星位定式（基本型）、小目定式入门、三线与四线、围空与取势、简单手筋（挖、点、刺）、死活进阶（刀把五、梅花五） |
| 中级 | 2级-1级 | 12 | 布局流派、开局思路、棋子配合、中盘战斗思路、手筋综合（滚打包收）、定式变着 |
| 中高级 | 1段-3段 | 10 | 定式深度理解、中盘攻防计算、死活提高型（聚三、直四）、手筋复杂型、官子基础 |
| 高级 | 4段-5段 | 8 | 大局观培养、复杂死活（老鼠偷油等）、高级手筋、综合复盘能力、官子精度的判断 |

**辅线：主题专题（可独立学习或横向复习）**

| 专题 | 题目数 | 说明 |
|------|--------|------|
| 吃子技巧 | 50+ | 征吃、枷吃、倒扑、接不归、双吃等 |
| 死活基础 | 50+ | 基础死活型及其应法 |
| 手筋练习 | 50+ | 攻击与防守手筋 |
| 布局专题 | 30+ | 布局选择、配置思路 |
| 中盘战术 | 40+ | 战斗策略与判断 |
| 官子收束 | 40+ | 官子大小、收束顺序 |

#### 3.2.2 教学内容形态

| 形态 | 适用场景 | 说明 |
|------|----------|------|
| 知识点讲解 | 理论学习 | 图文 + 动画演示棋理，配语音解说 |
| 互动引导 | 实战前哨 | 棋盘一步步带练，突出关键判断点 |
| 动手练习 | 知识巩固 | 学生在棋盘上实操，AI 即时反馈正误 |

#### 3.2.3 教学流程

```
知识点展示 → 引导练习 → 学生动手 → AI 评价 → 通过 / 复习
                                    ↓
                              80% 正确率通过
                              否则返回重新学习
```

每个知识点配 3-5 道练习题，通过后解锁下一知识点。

#### 3.2.4 教学路径

```
10级 ──┬─── 知识点 1-8 ────── 升级考试 ──► 8级
8级  ──┬─── 知识点 9-20 ───── 升级考试 ──► 6级
6级  ──┬─── 知识点 21-32 ──── 升级考试 ──► 3级
3级  ──┬─── 知识点 33-44 ──── 升级考试 ──► 1级
1级  ──┬─── 知识点 45-54 ──── 升级考试 ──► 1段
1段+ ──┴─── 知识点 55-62 ──── 升级考试 ──► 5段
```

### 3.3 练习模块（Practice）

#### 3.3.1 练习类型

| 类型 | 开关默认 | 说明 |
|------|----------|------|
| 吃子题 | 开启 | 征吃、枷吃、倒扑等 |
| 死活题 | 开启 | 基础到复杂死活型 |
| 手筋题 | 开启 | 攻防手筋综合 |
| 布局题 | 关闭 | 布局选点与配置 |
| 官子题 | 关闭 | 官子大小判断 |
| AI 变路 | 开启 | 完成后展示 AI 推荐的其他走法 |

所有类型均可独立开关，用户自定义练习组合。

#### 3.3.2 练习功能

| 功能 | 说明 |
|------|------|
| 每日挑战 | 每天5道题，按当前棋力自动抽取 |
| 错题本 | 答错题目自动收录，专项复习 |
| 能力图谱 | 雷达图展示各类型掌握程度 |
| AI 分析 | 完成后查看 AI 思路与推荐走法 |
| 题目收藏 | 收藏有价值的题目 |

#### 3.3.3 难度适配

- 练习题难度自动匹配用户当前棋力等级
- 正确率 > 80% 时，下次提升题目难度
- 正确率 < 50% 时，下次降低题目难度
